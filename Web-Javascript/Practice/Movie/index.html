장르 : 애니메이션
상영 시간 : 112분
국내 등급 : 15세 관람가

“이제 비는 그치고, 우리의 세상이 빛나기 시작할 거야”
비가 그치지 않던 어느 여름 날, 가출 소년 ‘호다카’는 수상한 잡지사에 취직하게 되고 비밀스러운 소녀 ‘히나’를 우연히 만난다.
“지금부터 하늘이 맑아질 거야”
그녀의 기도에 거짓말같이 빗줄기는 멈추고, 사람들의 얼굴에 환한 빛이 내려온다.
하지만, 맑음 뒤 흐림이 찾아오듯 두 사람은 엄청난 세계의 비밀을 마주하게 되는데…
